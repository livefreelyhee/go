# 면접 질문 관리 시스템

면접 준비를 위한 질문 및 답변 관리 웹 애플리케이션입니다. 기업별, 폴더별로 질문을 체계적으로 관리하고 실전 연습을 할 수 있습니다.

## 주요 기능

### 📁 기업 및 폴더 관리
- **기업 관리**: 기업별로 질문을 분류하여 관리
  - 기업 추가, 수정, 삭제, 복제
  - 기업 간 전환 및 드래그 스크롤
  - 우클릭 메뉴 지원
- **폴더 관리**: 기업 내에서 폴더로 질문 분류
  - 폴더 추가, 수정, 삭제
  - 폴더별 필터링
  - 드래그 스크롤 지원
  - 폴더 스코프 관리 (전체/기업별)

### 📝 질문/답변 카드 관리
- **카드 기능**:
  - 질문과 답변 작성 및 수정 (인라인 편집)
  - 줄바꿈 지원 (Enter 키로 줄바꿈)
  - 카드 선택, 고정, 드래그 앤 드롭
  - 카드 상세 보기 (전체 화면 모달)
  - 카드 이동, 복사, 삭제
  - 카드 복원 (휴지통 기능)
  - 더블클릭으로 상세보기
  - 카드 선택 해제 기능
- **정렬 기능**:
  - 기본 정렬 (고정 카드 우선)
  - 랜덤 정렬
  - 가나다순 정렬
  - 글자수 정렬 (질문+답변 글자수 합산, 작은 것부터)

### 🎯 실전 연습 모드
- **연습 설정**:
  - 완전 랜덤 모드
  - 폴더별 개수 설정 모드
  - 폴더별 최대 개수 제한
- **연습 기능**:
  - 스톱워치 (일시정지, 리셋, 자동 재생)
  - 다음 질문 자동 이동 (스톱워치 자동 리셋)
  - 질문만 보기 모드 지원
  - 전체 화면 모달
  - 키보드 단축키 지원

### 🖨️ 인쇄 기능
- 현재 필터된 카드 인쇄
- 질문만보기 모드 지원 (질문만 인쇄)
- 3열 그리드 레이아웃
- 인쇄 최적화 스타일
- 정렬 모드 반영

### 🗑️ 휴지통
- 삭제된 카드 복원
- 영구 삭제
- 휴지통 비우기
- 기업/폴더 삭제 시 카드 자동 이동

### 📋 기타 기능
- **일괄 추가**: 여러 질문을 한 번에 추가 (빈 줄 자동 제거)
- **Undo/Redo**: 작업 취소 및 재실행 (토스트 알림)
- **키보드 단축키**: 빠른 작업 수행
- **우클릭 메뉴**: 컨텍스트 메뉴 지원 (우클릭 위치에 표시)
- **자동 저장**: localStorage를 통한 데이터 저장
- **드래그 스크롤**: 기업 탭 및 폴더 섹션 드래그 스크롤
- **스크롤바 고정**: 스크롤바 유무와 관계없이 레이아웃 고정

## 사용 방법

### 기본 사용법

1. **기업 추가**: 상단 기업 탭에서 `+` 버튼을 클릭하여 새 기업 추가
2. **폴더 추가**: 하단 폴더 섹션에서 `+` 버튼을 클릭하여 새 폴더 추가
3. **카드 추가**: 우측 상단 `+` 버튼을 클릭하여 새 카드 추가
4. **질문/답변 작성**: 카드의 질문 또는 답변 영역을 클릭하여 편집
   - Enter 키로 줄바꿈
   - 다른 곳 클릭 시 자동 저장
5. **카드 선택**: 카드의 빈 공간을 클릭하여 선택 (복수 선택 가능)
6. **카드 고정**: 카드를 우클릭하여 "고정" 메뉴 선택
7. **카드 이동**: 드래그 앤 드롭으로 카드 순서 변경 (기본 정렬 모드)
8. **카드 상세보기**: 카드를 더블클릭하여 상세보기 모달 열기

### 실전 연습

1. 상단 "실전 연습" 버튼 클릭
2. 연습 모드 선택 (완전 랜덤 또는 폴더별 개수 정하기)
3. 폴더별 개수 설정 (선택한 경우)
   - 최대 개수 제한 자동 적용
   - 토스트 알림으로 제한 안내
4. "시작" 버튼 클릭
5. 질문을 읽고 답변을 생각한 후 "다음" 버튼으로 다음 질문으로 이동
6. 일시정지/재생 버튼으로 타이머 제어
7. 리셋 버튼으로 타이머 초기화

### 인쇄

1. 상단 "인쇄" 버튼 클릭
2. 현재 필터된 카드들이 인쇄 미리보기로 표시
3. 질문만보기 모드일 경우 질문만 인쇄
4. 브라우저 인쇄 대화상자에서 인쇄 실행

### 정렬 모드

- **기본**: 고정된 카드가 먼저 표시되고, 그 다음 순서대로 표시
- **랜덤**: 카드를 무작위로 섞어서 표시
- **가나다**: 질문을 가나다순으로 정렬
- **글자수**: 질문과 답변의 글자수 합산하여 작은 것부터 정렬

### 키보드 단축키

- `Delete`: 선택된 카드 삭제
- `Ctrl+A` (또는 `Cmd+A`): 현재 필터된 모든 카드 선택
- `Escape`: 모달 닫기
- `←` / `→`: 카드 상세보기에서 이전/다음 카드 이동

## 데이터 저장

- 모든 데이터는 브라우저의 localStorage에 저장됩니다
- 브라우저 데이터를 삭제하면 저장된 데이터도 함께 삭제됩니다
- 데이터 백업을 원하는 경우 인쇄 기능을 사용하거나 브라우저 개발자 도구에서 localStorage 데이터를 복사하세요
